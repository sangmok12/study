## IAM 계정 생성

서비스 찾기에서 IAM을 검색한다.
좌측 엑세스 관리에서 그룹 선택,
새로운 그룹 생성  그룹 명(EC2_Group) 정하고 정책유형에  EC2Full 검색.  AmazonEC2FullAccess   체크 후 다음.

좌측  사용자 누르고
AWS 매니져콘솔  체크하지말고  생성

그룹에 사용자 추가 하고
액세스 키 만들기,
Csv 다운로드 



## 역할 만들기

##  인스턴스 생성

서비스에  EC2 검색,  인스턴스 시작
이름(내 서버) 정하고,  키 페어(AWS_EC2) 생성하고 시작
다 되면 모든 인스턴스 보기 클릭
좌측에  탄력적 IP 클릭
탄력적 IP 주소 할당  누르고  할당 
할당 되었습니다 오른쪽에  이 탄력적 IP 주소 연결  클릭
인스턴스에  내 서버 선택하고 연결
