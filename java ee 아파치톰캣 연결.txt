이클립스 엔터프라이즈 자바 엔 웹 디벨로퍼  설치

New   다이나믹 웹 프로젝트 생성 

오른쪽 위 커피모양 JAVA EE  여야 함   그게 JSP 환경
그 왼쪽 오픈 퍼스펙티브 누르면  다른 환경도 열 수 있음 ( 자바 EE 닫겨도 이거로 열면 됨 )


아파치 톰캣 (버전 높을수록 좋음) 압축 풀어서  C 드라이브에 위치하기.

이클립스 윈도우 >  프리퍼런스 > 서버 > 런타임 인바이런먼트   ADD > 아파치 버전맞게 추가. > 브라우즈 눌러서 폴더위치 설정

프리퍼런스 웹에 들어가서  JSP JAVA HTML  UTF-8 확인.   제네럴 > 워크스페이스 도 확인


src > main > wepapp 안에 작업할 파일.jsp파일 만들기

작업화면에서 오른쪽 눌러서 프리퍼런스에서 글자크기 조절 가능

윈도우 > 프리퍼런스 > 서버 > 런타임 인바이러먼트 > 애드 아파치  버전 
왼쪽 목록창에서 오른쪽 버튼 눌러 빌드패스 > 컨피규어 빌드패스 > 라이브러리 > 모듈패스 누르고 > Add Library > 서버런타임 > 아파치 > Finish\
